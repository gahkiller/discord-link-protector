# 🛡️ Discord Link Protector PC

A powerful Chrome extension that protects Discord users from **scam links, malware, phishing, and unsafe server invites** in real time.

---

## 🚀 Features

### 🔴 Real-Time Link Protection
- Scans messages instantly as they appear  
- Flags suspicious links by turning them **red**  
- Works across chats, servers, and Discover page  

### 🧠 Smart Scam Detection
- Detects:
  - Fake giveaways (free money, Nitro, etc.)
  - Phishing attempts
  - Referral scams
- Uses advanced text normalization to catch bypass tricks (like `fr33 m0ney`)

### ⚠️ Malware & Virus Detection
- Flags dangerous links as:
  > ⚠️ **Virus detected**
- Detects:
  - Malware
  - Trojans
  - Token grabbers
  - Loggers / stealers  

### 🔗 Suspicious Link Detection
- Blocks:
  - URL shorteners (bit.ly, b1t.ly, etc.)
  - IP grabbers (grabify, iplogger)
  - Scam / crypto / gambling sites  
- Also detects **unknown suspicious links**, not just listed ones  

### 🧩 Server Protection
- Scans Discord **Discover page**
- Highlights scammy or risky server names  

### 🚫 Block / Unblock System
- One-click **Block** button  
- Blocked links:
  - Turn gray  
  - Become unclickable  
- Easily **unblock anytime**  

### ⚡ Real-Time Monitoring
- Automatically scans:
  - New messages  
  - Edited messages  
  - Newly loaded content  

### 🎨 Clean UI
- No intrusive popups  
- No screen overlays or borders  
- Simple inline warnings  

---

## 🛠️ Installation

1. Download or clone this repository  
2. Go to `chrome://extensions/`  
3. Enable **Developer Mode**  
4. Click **Load unpacked**  
5. Select the extension folder  

---

## 🔒 Permissions

- `storage` → Saves blocked links  
- `scripting` → Runs protection script  
- `tabs` → Supports advanced scanning  

---

## ⚠️ Disclaimer

This extension provides **strong protection**, but no system is perfect.  
Always stay cautious when clicking unknown links.
